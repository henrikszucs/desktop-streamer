    About Desktop Streamer

    Version: 1.0.0

    Desktop Streamer is an open source project that allows you to access your desktop remotly through a simple web interface. It is built with Electron and Node.js.

    Missing features •Nothing missing, all features are supported. •Auto launch: not supported •System tray: not supported •System audio share: only manual share •System audio share: not supported •Screen share: only manual select •Low latency play: not supported •Control share: not supported
